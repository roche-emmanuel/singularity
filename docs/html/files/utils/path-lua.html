utils.path

Path manipulation and file queries package.  This is modelled after Python’s os.path library (10.1); see @{04-paths.md|the Guide}.

normalize

function Class.normalize(path)

Normalize the given path.  This will ensure that paths such as: A//B, A/./B, A\B\, and A/foo/../B all become A/B.  Note that this method is also convert the windows separator to the normalized unix separator The path can then be converted back to windows format.

Parameters

path{string} The path to normalize.

Returns

The normalized path.
