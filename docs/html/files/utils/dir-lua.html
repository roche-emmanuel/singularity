getFiles

function dir.getFiles(dir,
mask)

Return a list of all files in a directory which match the a shell pattern.  Note that the dir argument is prepended to the file paths returned.

Parameters

dirA directory.  If not given, all files in path.root_dir directory are returned.
maskA shell pattern.  If not given, all files are returned.

Returns

An <std.Vector> of files.  Raise an error if dir or mask are not strings.

getDirectories

function dir.getDirectories(dir)

Return a list of all subdirectories of the directory.

Parameters

dir{string} A directory.  If not given, utils.path.root_dir is used.  Raise an error if not a valid directory.

Returns

An <std.Vector> of directories.
