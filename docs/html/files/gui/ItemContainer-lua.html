gui.ITemContainer

Specific container implementation describing multiple items in a table.  The items are stored under integer keys to maintain a proper ordering.

This class inherits from gui.Container.

setIDFunction

function Class:setIDFunction(func)

Set the function to use to retrieve an ID from a given item.  By default, the function used will simply retrieve the name of the item from the “name” element of the item table.

Parameters

funcThe funciton to use to retrieve the item ID.  The function prototype should be ID func(item);
